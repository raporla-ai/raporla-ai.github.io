Raporla

Innovative solution for radiologists to write reports using voice recognition

Project Information

Raporla is an AI-driven solution that enables radiologists to create reports efficiently by using their voice. The system utilizes speech-to-text technology, fine-tuned specifically for Turkish doctors, to automate and streamline the report creation process.
